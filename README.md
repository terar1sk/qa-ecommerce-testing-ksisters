# QA Testing Portfolio – Ksisters Website

## Project Overview
This project is a manual QA testing practice for the website [ksisters.sk](https://ksisters.sk/).
This project is part of my QA portfolio.
The goal of this project is to improve testing skills, understand real user behavior, and improve how to document bugs and test results.

## Scope of Testing
The following parts of the website were tested (this list will be extended during testing):
* Search functionality
* Product pages
* Cart functionality
* Navigation (header and menus)
* Login form

## What Was Done
* Created a test plan
* Defined test scenarios
* Performed exploratory testing
* Identified and documented bugs
* Added screenshots as visual evidence
