# Test Plan - Ksisters Website

## Objective
The goal of this testing is to check the main functionality of the [ksisters.sk](https://ksisters.sk) website and find possible issues in user experience.

## Scope
The following parts of the website were tested:
* Search functionality
* Product pages
* Cart functionality
* Navigation (header and menus)
* Login form

## Out of Scope
The following parts were not tested:
* Payment system
* Backend logic
* Performance testing

## Test Approach
Testing was done using black-box approach.
The system was tested from a user perspective by interacting with different parts of the website.
Different input types and user actions were used to check system behavior.

## Test Types
The following testing types were used:
* Manual testing
* Exploratory testing
* Functional testing
* UI testing

## Tools
* Google Chrome (latest version)
* Chrome DevTools
* GitHub (for documentation)

## Environment
* OS: Windows 11 Home
* Browser: Google Chrome (latest version)

## Risks
* No access to backend logic
* Expected behavior is based on assumptions
* Some features may depend on region or user state
